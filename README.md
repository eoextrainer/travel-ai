# Travel AI Project

Complete project structure for the EOEX Travel AI Agent course.

This repository contains a complete workshop curriculum and implementation plan for building a Travel AI Agent that simulates the workflow of a professional travel agency.

- Audience: Non-technical participants
- Duration: 2 hours
- Deliverables:
  - Step-by-step workshop curriculum (`docs/workshop-curriculum.md`)
  - AI prompt optimized for Copilot, ChatGPT, Claude, Gemini (`prompts/workshop-prompt.md`)
  - Implementation plan (`docs/implementation-plan.md`)
  - Technical stack and dependencies (`docs/technical-stack.md`)
  - Git workflow and version control guide (`docs/git-workflow.md`)
  - Portable aggregated markdown (`docs/portable.md`)
